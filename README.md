# StickyNotes

* メモ/付箋作成アプリです。

## 使い方

* 付箋作成ボタンで付箋を作成。
* 付箋左側で削除と色変更。
* ノートエリアにテキストを記述。
* ノート右上の「X」ボタンでノートを削除。

## 使用技術

* HTML
* CSS
* JavaScript
* Firebase
